# 1 页面布局
## 1.1 问题：高度已知，三栏布局，左右栏宽度都为300px，中间自适应
### 布局方式

* float
  * 优点：兼容性好
  * 缺点：浮动会脱离文档流，会影响`.layout`的兄弟元素，需要做好清除浮动；`.center`高度比两边高时，内容会流动到两边；`.layout`的高度同`.center`高度，`side高度`可能会超出`.layout`

* 绝对定位
  * 优点：兼容性好，布局快捷
  * 缺点：元素脱离文档流，`.layout`高度不能撑开

* table
  * 优点：兼容性很好
  * 缺点：高度会同时变化，语义化问题

* flex
  * 优点：CSS3的布局方式，目前比较完美的方案
  * 缺点：兼容性

* grid
  * 优点：可以做更复杂布局
  * 缺点：兼容性还不够好
