* ls
```bash
ls # 列出文件
ls -l # 列出文件详细信息 权限 时间戳
```

* mkdir
```bash
mkdir dir1 # 创建目录

cd ~ # 到用户目录
```

* rmdir
```bash
rmdir dir1 # 删除目录，不能删除有文件到目录
```

* cp
```bash
cp dir1/original dir2/duplicate # 复制文件并重命名
```

* mv
```bash
mv dir1/original lab2/original
mv dir1/original lab2/rename # 移动并重命名
```

* rm
```bash
rm file1 ## 删除文件
rm -r dir1 ## 删除文件夹和其中到文件
```
